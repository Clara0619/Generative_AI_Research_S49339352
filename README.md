# Generative AI Research Project

This repository contains materials for the research project **"Using Generative AI Tools - Boon or Bane"**. It is used to organise research documents, data files, analysis scripts, reports, and collaboration records.

## Repository Structure

- `Literature Review documents/`: journal articles, conference papers, books, and newspaper articles.
- `quantitative analysis/`: survey data, survey questions, Python scripts, CSV/Excel files, and survey analysis reports.
- `Qualitative analysis/`: interview transcripts, interview protocols, consent forms, analysis reports, and data visualisations.
- `Drafts and Reports/`: draft research proposals, conference papers, and final reports.
- `Additional Materials/`: information sheets, photos, and other media files.
- `Project_logbook.txt`: records project updates and team collaboration activities.

## Collaboration Guidelines

Team members should use clear and consistent file names so that files are easy to identify and manage. When making changes, collaborators should create a new branch, edit the relevant files, commit their changes with a meaningful message, and create a pull request for review. Sensitive files, such as consent forms, interview transcripts, and raw participant data, should be handled carefully and only shared with authorised team members.
